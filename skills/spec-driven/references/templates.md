# 4ドキュメント テンプレート

## PLAN.md テンプレート

```markdown
# PLAN: [プロジェクト/タスク名]

## 背景
<!-- なぜこれをやるのか -->

## やりたいこと
<!-- 自由記述。箇条書きでも文章でもOK -->

## 参考情報
<!-- URL、既存コード、関連ドキュメントなど -->
```

## SPEC.md テンプレート

```markdown
# SPEC: [プロジェクト/タスク名]

## ゴール（完了条件）
- [ ] 条件1
- [ ] 条件2

## 制約・前提条件
-

## 入出力の具体例
### 入力
### 出力

## 設計メモ
<!-- 技術的な判断・トレードオフなど -->

## 合意状況
- [ ] 未合意
- [ ] 合意済み
```

## TODO.md テンプレート

```markdown
# TODO: [プロジェクト/タスク名]

## タスク
- [ ] タスク1
- [ ] タスク2
- [ ] タスク3

## 完了レビュー
<!-- タスク完了後に記入 -->
- 動作確認:
- 残課題:
```

## KNOWLEDGE.md テンプレート

```markdown
# KNOWLEDGE: [プロジェクト/タスク名]

## 学習記録

| 日付 | カテゴリ | 内容 | 解決策 |
|------|----------|------|--------|

## ハマりポイント
<!-- 詳細な記録が必要な場合はここに追記 -->

## ユーザーからの修正
<!-- Claudeが修正を受けた内容を記録 -->
```
